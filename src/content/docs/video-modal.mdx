### Props

| Prop             | Type                                                                                                                                                      | Default             | Description                                                                          |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------- | ------------------------------------------------------------------------------------ |
| `animationStyle` | `"from-bottom"` \| `"from-center"` \| `"from-top"` \| `"from-left"` <br/> \| `"from-right"` \| `"fade"` \| `"top-in-bottom-out"` \| `"left-in-right-out"` | `"from-center"`     | Determines how the modal animates in/out using Framer Motion.                        |
| `videoSrc`       | `string`                                                                                                                                                  | `—`                 | URL of the embedded video (e.g., YouTube embed link). Must support iframe embedding. |
| `thumbnailSrc`   | `string`                                                                                                                                                  | `—`                 | URL for the thumbnail image shown before the video is played.                        |
| `thumbnailAlt`   | `string`                                                                                                                                                  | `"Video thumbnail"` | Alternative text for the thumbnail image (for accessibility).                        |
| `className`      | `string`                                                                                                                                                  | `—`                 | Optional CSS class to apply to the outermost wrapper of the component.               |
